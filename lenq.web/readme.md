# common.js通用函数的封装

>	用来解决平常开发，经常用到的功能

vendor下的common.js：

```javascript
	Common:
		now()	// 获取时间戳
		share() // 分享
		doAjax() // ajax的简单封装
		adaption() // 弹窗自动适应窗口
		limitPhone() // 限制只能输入数字、而且限制只能输入11位有效数字
		placeholder() // 兼容IE6-8 的placeholder的功能
	dialog:
		open()  // 弹窗的打开
		close() // 弹窗的关闭
 ```
